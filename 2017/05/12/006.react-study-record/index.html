react学习记录

2017.01.08

  1. 在iOS上使用http链接的图片地址可能不会显示，参见这篇说明修改。

  2. 需要的组件都要在顶部引用。

  3. 不能直接在组件外面写样式，需要显示传递，样式名按骆驼命名法书写
    顺序声明和使用style属性，以借鉴CSS中的“层叠”做法（即后声明的属性会覆盖先声明的同名属性）

  4. Flexbox布局的说明 http://reactnative.cn/docs/0.40/layout-props.html
